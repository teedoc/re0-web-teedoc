## build locally

* Install teedoc

```
pip3 install teedoc
```

* Get site source files

```
git clone https://github.com/teedoc/re0-web-teedoc re0-web-teedoc
```


* Install plugins

```
cd re0-web-teedoc
teedoc install
```

* build or serve

build and serve locally:
```
teedoc serve
```

then visit [http://127.0.0.1:2333](http://127.0.0.1:2333)


only build:
```
teedoc build
```
